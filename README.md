# M-dulo-4.-Ejercicios-Javascript.-Curso-Sence-FET # README — Resumen Ejecutivo del Proyecto
## Fundamentos de JavaScript - Formulario de Ejercicios

### 1. Propósito General del Proyecto
Este proyecto ha sido desarrollado como una herramienta educativa y demostrativa en el marco del **Programa de Formación SENCE: Curso Front End Trainee**. Su objetivo principal es materializar los conceptos fundamentales de la programación con JavaScript (tales como variables, estructuras condicionales, bucles iterativos, funciones modulares, manipulación de objetos y control de colecciones) y conectarlos directamente con una interfaz gráfica interactiva en el navegador a través de la API del DOM (*Document Object Model*)

---

### 2. Componentes del Proyecto y Arquitectura
El sistema se compone de dos archivos principales:
1. **`ejercicios.html` (Capa de Presentación):** Define la estructura semántica de la página mediante etiquetas HTML5 y contiene los estilos CSS que garantizan una experiencia de usuario limpia, ordenada y reactiva
2. **`script.js` (Capa de Lógica e Interacción):** Contiene la inteligencia de la aplicación. Está dividido en:
   * **Parte A (Lógica Pura):** Algoritmos aislados e independientes del navegador que procesan datos brutos
   * **Parte B (Interactividad DOM):** Manejadores de eventos que capturan las interacciones en la UI y actualizan la vista en tiempo real de forma segura

---

### 3. Funcionalidades Implementadas
* **Módulo 1: Saludo Personalizado:** Limpieza de strings (`trim`) y validación de campos vacíos con respuestas personalizadas y gestión dinámica de errores
* **Módulo 2: Comparador:** Evaluación matemática de variables numéricas determinando relaciones de mayor, menor o igualdad
* **Módulo 3: Arreglos y Bucles:** Comparativa práctica del flujo de control iterativo empleando estructuras `for` y `while`, además de un algoritmo de filtrado condicional inductivo
* **Módulo 4: Calculadora Modular:** Implementación de funciones con una sola responsabilidad unificadas por un control de flujo `switch`, incluyendo la prevención explícita de la indeterminación por división cero
* **Módulo 5: Objetos y Colecciones:** Uso avanzado de métodos funcionales modernos (`.map`) aplicados sobre arreglos de objetos para la mutación y despliegue estético de datos
